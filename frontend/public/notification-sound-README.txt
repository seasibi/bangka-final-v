NOTIFICATION SOUND SETUP
========================

Please add a notification sound file named "notification-sound.mp3" to this public directory.

You can download a free notification sound from:
- https://freesound.org/
- https://www.zapsplat.com/
- https://mixkit.co/free-sound-effects/notification/

Recommended specifications:
- Format: MP3
- Duration: 0.5 - 2 seconds
- File size: < 100KB
- Type: Pleasant notification chime or alert sound

The sound will be played when a boundary violation notification is received.
